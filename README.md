## 联系人随机头像

- 需要下载相关 APP，Data Jar
- 支持Surge正式版的模块分类
- 支持模块下载到本地文件夹（添加模块的方式：模块名：模块链接）
- 支持模块的增删和自动重启 Surge

## Surge 模块管理器

- 需要下载相关 APP，Data Jar
- 支持Surge正式版的模块分类
- 支持模块下载到本地文件夹（添加模块的方式：模块名：模块链接）
- 支持模块的增删和自动重启 Surge

## Surge 模块管理器（无依赖版）

- 内置部分自用模块
- 支持Surge正式版的模块分类
- 支持模块下载到本地文件夹（添加模块的方式：模块名：模块链接）
- 支持模块的增删和自动重启 Surge
